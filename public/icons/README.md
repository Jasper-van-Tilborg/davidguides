# PWA Icons

Plaats hier de volgende icon bestanden:

- `icon-192.png` (192x192 pixels)
- `icon-512.png` (512x512 pixels)

## Hoe icons te genereren:

1. Ga naar https://favicon.io
2. Kies een optie:
   - **PNG/JPG to favicon** - Upload je logo
   - **Text to favicon** - Genereer van tekst
   - **Emoji to favicon** - Gebruik een emoji
3. Download het package
4. Gebruik:
   - `android-chrome-192x192.png` → hernoem naar `icon-192.png`
   - `android-chrome-512x512.png` → hernoem naar `icon-512.png`
5. Plaats beide bestanden in deze folder

## Apple Touch Icon

Plaats `apple-touch-icon.png` (180x180 pixels) in de `public/` folder (niet in icons/).

## Icon Design Tips

- Gebruik een simpel, herkenbaar design
- Zorg voor goede contrast (werkt op lichte en donkere achtergronden)
- Houd belangrijke elementen gecentreerd (veilige zone: middelste 80%)
- Test op echte devices om te zien hoe het eruit ziet
